# online-course-management-system

online course management system

Tech-stack :

1. Java
2. Servlet
3. JSP
4. MySQL
5. Maven
6. JSTL
7. Eclipse

Features :

1. Admin (login needed)
 1. Assign Course
 2. Teacher List View
 3. Student List View
2. Teacher (login needed)
  3. Course List View
3. Course Registration (login needed)
3. Student (login needed)
  1. Course Registration
  2. Course List View
