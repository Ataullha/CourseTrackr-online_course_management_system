# online-course-management-system

online course management system

Tech-stack :

1. Java
2. Servlet
3. JSP
4. MySQL
5. Maven
6. JSTL
7. Eclipse

Features :

- Admin (login needed)
  - Assign Course
- Teacher List View
  - Student List View
- Teacher (login needed)
   - Course List View
- Course Registration (login needed)
- Student (login needed)
   - Course Registration
   - Course List View
